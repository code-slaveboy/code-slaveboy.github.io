想说的话

鸣谢

  • 感谢我的技术支持者，帮助我搭建了我的第一个网站，从搭建的过程中我也学到了很多。当然这只是一个开端，日后还会不断学习。

目前的状况

  • 目前还在准备考研，乘着最近休息之余，想着搭建自己的网站，之前以为很难的但是搭建之后，我觉得没有太难。果然，还是得亲自动手才知道难不难

  • 不过，目前还在备考研，应该不会有太多的时间来管理和维护我的网站。我的网站还没有买域名（因为vercel进不去），也没有买服务器，单纯靠github。浏览的时候会有些许卡顿，日后有时间了我就来解决这个问题吧。

  • 商务英语专业在读练习生，练习时长两年半。向往计算机上发展一下，提升一下个人的竞争力。毕竟是自己的兴趣，学习这些东西不需要花钱，不过有些费时间。有机会得学一学HTML和JavaScript，不会的和不懂的太多了，目前我只学了C语言和数据结构，并没有太多的突破。期待有一天我也能在我的blog上给大家分享技术，这个愿望先放着来了，我定会努力实现，感谢各位支持

  • 网站的背景图片是网上找的，侵权删，请联系我，点击主页下面的邮箱。

搭建网站的过程中所遇到的问题

  • 最大的问题
    我将出错信息放在下边吧
出错步骤 解决方法
hexo init git config —global http.sslBackend schannel

错误信息：
fatal: unable to access ‘https://github.com/hexojs/hexo-starter.git/‘: error setting certificate file: D:\Git\mingw64\ssl\certs WARN git clone failed. Copying data instead

  • 就这个错误我才网上找了好久才解决，初始化的时候，那个git clone一定得成功不然到最后一步hexo g的时候就会报错，不会弹出github的登陆框，就无法在服务器部署你的网站。

  • 若果你想拥有自己的的个人博客不知道怎么搭建的，就去看大佬的视频吧，请移步至Fomalhaut🥝，详见大佬的hexo博客搭建教程。有什么不懂的别问我，我真的不会，谢谢大家。

建站初衷

  • 既然是初次搭建，那么我就在这里写一下我的初衷吧。

    其实，我早就想搭建属于自己的网站，不过我把这些东西想的很难，加上自己很懒，想着要搭建，但是一直没有动手。前天考完试，我想着休息一下，于是心血来潮，开始照着大佬的视频搭建和美化我自己的blog。

    博客其实没什么人会看，大多时候是自己看的，因此我搭建这个网站来记录我的学习过程和成长过程，QQ空间和朋友圈似乎不太适合发布学习的内容。没有人见证我的成长那就我自己来吧。这个网站专门用来记录我的学习，当然也会写一下我的小情绪，因为我的朋友太少了，而且有些负面情绪我不太想向我的朋友倾诉，也只会给人徒增烦恼。我倒是希望我的朋友能快乐一点，看到我积极的一面，希望我的朋友也能开心，积极阳光。我真的不希望任何人为了我操心。

    写了这么多，那我也来讲点个人故事吧，你要是能点进来就吃点瓜🥝吧哈哈。💊认真看嘞,你不看也没关系

    我从上了大学开始就开始摆烂了，超喜欢玩阴阳师。有天在游戏里认识了一个女孩子，于是开始无话不谈地聊，聊游戏，聊理想，聊日常琐事，大概持续了一年吧，人家上的学校好像还不如我，但是似乎比我努力好多倍，我想着光玩阴阳师成不了事，于是开始努力学。但是我发现大学的课堂其实没学到什么，而且本专业开的水课实在是太多了，而且也不怎么喜欢听，但还是听课还是听了个大概。(我们专业的老师第一节课就说期末的事情，合着就跟我上课就是为了期末一样。都说理论能知指导实践，但是我们专业都没有实践过。如果有还没上大学的友友，要是喜欢外语的话，最好是作为第二选择，因为感觉没有学到太多有帮助的东西，到后面二年级可以辅修)。那时候也没有天天玩游戏了，但是游戏时间依旧很长吧。我周围的人也是这样，基本上没有什么人学习。我越来越觉得不对劲，越玩就越是焦虑。

    后来我选择辅修计算机科学与技术，渐渐就没有太多时间去维护我的阴阳师了。但是这期间一直有和那位女生联系。聊了大半年，感觉有些喜欢，但是没有说，感觉自己配不上，而且嘛好像圈子也没有人家那么大。虽然这样，平时还是会腻歪。聊了一年半，她有天说要给我告白，我以为开玩笑哈哈哈哈，然后真的就告白啦。真是big胆，连我照片都没有看过。我感觉她好勇敢，我自己本来也蛮喜欢的。那时候本来计划毕业就去实习。那小子说她准备考研。于是我觉得我也想考。（你不知道我在写这个的时候笑的有多开心呐）我跟她说要考计算机的研究生。当时还商量着要见面，在十月份呜呜呜。

    刚确定关系的时候，我俩狠狠地聊，记得当时打电话能聊一整天。但是时间长了，话就没那么多了，慢慢冷淡了。后面就和平常一样啦，有时间就聊（当时她要考教资）

    慢慢到了要见面的日子（国庆），超级期待，兴奋了好几个晚上。

    我俩刚见面，她说我该减肥了，我说很胖吗，她说我有两个她那么宽了哈哈。她是晚上来的又坐火车又坐地铁啦，当时又坐为了省钱，订的双床房我俩住一个房间（双床房），头回和女生住一屋，紧张死了。第二天一起去了罗中立美术馆，重庆动物园，和解放碑。感觉她出去走一点也不会累。想牵她的手手，但是被拒绝啦呜呜呜呜。和她一起去玩是真的很能瘦。和她去玩我居然感冒了，老天真是不眷顾我。她还把我带去她们学校玩。我在那边玩了三天，骑电瓶车带着女朋友的感觉真的好。我是真的喜欢在一起的那种感觉，就算什么都不做，我感觉也会很开心吧。和她一起出去就完了一天。大概是奔波太久了，太累了，本来商量好还要去别的地方玩，结果累趴下了，就没去，然后我一个人的时候就在那边狂炫。

    回来之后没多久，她说想分，说我有些幼稚，因为跟她分别的时候我快要哭了。我说太遗憾了，其实我放不下，我明白自己没有那么优秀，也没有实力去扭转任何人的心❤️，很难接受，但是还是同意分了。我一个人在操场上走了三个小时还是四个小时，像是丢什么东西，我就是停不下来，然后就一直走走走走。我倒是知道网恋加异地很难啦，本想接受这个事实，但是那小子说她后悔了，我太高兴了，顿时所有的迷茫和奇奇怪怪的东西就消散了。

    但是后来话越来越少了，也不怎么联系，就两三天才联系，而且也没什么话说了。有天她被人讨论（精神+物理攻击，前社团的奇葩吧），她第二天发QQ直接挂社团和攻击她的人了，感觉委屈她了，但是我不在她身边，真怕她打起来，就选择了妥协点的解决方法，双方都道歉了。我还是有点不满意，那些人有确实问题，过后还要在社团官方QQ阴阳，我不想追究这种事情就算了，没有管。

    后来没什么事，我问过她到底喜不喜欢我，她说不知道。我觉得没关系吧。我自己倒是很喜欢，不过有哪天真的想走就放她走吧，真不想让她不舒服。

    在元旦后的第几天，忘了是第几天，忘了忘了，就是说不想谈恋爱了，觉得谈恋爱，结婚这些东西会妨碍她吧，她说她和她堂姐聊天，还没有找到工作就结婚了。她给我讲了她堂姐我当时要期末考试，记得是考试的前一晚吧应该，挽留了，但是还是分了，也没有多卑微吧。哭了，在只有我一个人的教室里哭。可能是我讲话的方式让人有些窒息，也可能这种关系并不长久，好像似乎让她感觉愧疚。

    我看着她跟别人聊的开心其实挺羡慕的呜呜呜呜，真的好羡慕。但是我感觉我还是有点放不下。我想考过去，解决她想到的多有困难，和她在一起是一种从未有过的快乐的感觉。如果在考研期间要是听说她有新的男朋友了我估计我的❤️会碎吧。但是也得接受呜呜呜

    商务英语跨考计算机，其实希望不大吧。但是我还是要试试。不想考408，于是选择考自命题的院校，只考c语言和数据结构这两门。从2023年七月份我就开始学数学了，感觉好难，但是还是硬着头皮上了。但是大三上，课也挺多的，进度也很慢。现在一天学八九个小时，最近期末了，紧张死了。

  • 谢谢你能看到这里，我说完了，谢谢你，再见
